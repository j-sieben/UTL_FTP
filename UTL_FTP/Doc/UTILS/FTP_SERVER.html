Table FTP_SERVER

Table to store predefined FTP server connect data
Columns
Name Type Optional Default Comments
FTP_ID VARCHAR2(30)     Unique FTP server identifier. Primary key
FTP_HOST_NAME VARCHAR2(200) Y   IP address or DNS name of the FTP server. NOT NULL
FTP_PORT NUMBER Y   Port the FTP server listens at. NOT NULL
FTP_USER_NAME VARCHAR2(200) Y   Name of the user that connects to the server
FTP_PASSWORD VARCHAR2(80) Y   Password of the user that connect to the server
